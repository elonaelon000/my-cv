# Elona Tarja — Portfolio CV

A responsive personal CV and portfolio built with HTML, CSS, and JavaScript.

## Files

- `index.html` — page structure and content
- `style.css` — black and pink visual design
- `script.js` — mobile menu, scroll animation, active navigation, and cursor glow

## Run locally

Open `index.html` in a browser, or use the Live Server extension in VS Code.

## Publish with GitHub Pages

1. Push the files to the `my-cv` repository.
2. Open the repository on GitHub.
3. Go to **Settings → Pages**.
4. Under **Build and deployment**, choose **Deploy from a branch**.
5. Select the `main` branch and the `/ (root)` folder.
6. Click **Save**.

The page should become available at:

`https://elonaelon000.github.io/my-cv/`
